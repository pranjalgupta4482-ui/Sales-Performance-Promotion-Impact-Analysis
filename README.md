# Sales-Performance-Promotion-Impact-Analysis
End-to-end Sales Analytics Project using Excel, SQLite, Python &amp; Power BI
📊 Sales Performance & Promotion Impact Analysis
📌 Project Overview

This project analyzes sales performance and evaluates promotional impact to identify high-performing SKUs and revenue growth opportunities.

🎯 Problem Statement

Businesses often struggle to:

Identify top revenue-generating SKUs

Measure promotion effectiveness

Track Year-over-Year growth

Monitor monthly revenue trends

🛠 Tools & Technologies

Excel – Data Cleaning

SQLite – Data Extraction

Python – Data Preprocessing & Scatter Plot Analysis

Power BI – Data Modeling & Interactive Dashboard

📈 Key Insights

Identified Top 20 revenue-driving SKUs

Analyzed promotion impact on revenue

Measured YoY growth trends

Built executive-level KPI dashboard
